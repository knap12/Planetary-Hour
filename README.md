# Planetary Hour
Production Chrome extension that calculates and displays the current planetary hour using geolocation, sunrise/sunset timing, background automation, and a custom time-based rules engine.

## Features

- Real sunrise/sunset timing
- Geolocation integration
- Dynamic toolbar icon updates
- Automated background scheduling
- Persistent state management
- Payment/authentication support

## Tech Stack

- JavaScript
- Chrome Extensions
- Chrome APIs
- Geolocation API
- Async API integrations
- ExtensionPay
- Stripe

## Engineering Challenges Solved

- Service-worker lifecycle limitations
- Background update reliability
- Sequencing and time-boundary logic
- State consistency across restarts

## Planned Features

- Countdown until next planetary change
- Current planetary-hour window display
- Full daily planetary schedule
